Twitter Updates Widget

This is a great script for showing your latest tweets.

Features

With this widget, you can show your latest tweets on your website. Create your own widget with its great customizability.

  • Display tweets from a search, or a list or account(s)'s feeds!
  • Supports multi-account
  • A simple theme for minimalism-fans!
  • It's fully customizable
  • Auto-linking mentions, hashtags and urls
  • 1-minute-install!
  • Powered by jQuery
  • Choose from two themes, 7 color styles (blue, dark blue, green, red, gray, black&white, orange), 4 icon styles (twitter birds and logos), 3 effects (fade, slide, none)
  • Auto-scroller
  • Options to display the username, avatar, time, and arrows to scroll to the next tweet



Installation

After copying tuw-inc folder to your server; there are two steps for installation. Also you can look over examples in the examples folder

Step #1

Create a div (or any HTML element) to where you want to add the widget. Or just Copy and Paste the following code.

<div id='tuw_div'></div>


Step #2

Here's the last and important part: Creating our widget. You can add the simple version which is really short to add the widget with default settings or the advanced version for customization so you can create your own widget with many options!

Simple Installation Code
<!-- Setting "Twitter Updates Widget" Up -->
	<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.4.2/jquery.min.js"></script>

	<script type="text/javascript" language="Javascript" src="tuw-inc/jquery.tuw.js"></script>
	<link rel="stylesheet" href="tuw-inc/style.css" type="text/css" media="screen" />
	<script type="text/javascript" language="Javascript">
		$(document).ready(function(){
			$('#tuw_div').tuw();
		});
	</script>
<!-- This is it -->

Advanced Installation Code
<!-- Setting "Twitter Updates Widget" Up -->
	<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.4.2/jquery.min.js"></script>

	<script type="text/javascript" language="Javascript" src="tuw-inc/jquery.tuw.js"></script>
	<link rel="stylesheet" href="tuw-inc/style.css" type="text/css" media="screen" />
	<script type="text/javascript" language="Javascript">
		$(document).ready(function(){
			settings = {
				'username' : ['ebuyukkaya'],
				'simpletheme' : false, 	
				'updates' : 5, 				
				'list' : null,				
				'query' : null,				
				'color' : 'blue', 			
				'openbox' : true,	 		 
				'icon' : '1', 			
				'autonext' : false,				
				'duration' : 5000,					
				'linksinnewpage' : false,		
				'showavatar' : true,				
				'showusername' : true,			
				'showdate' : true,					
				'singlepostmode' : false,		
				'effect' : 'slide',	
				'defaultPicture' : 'http://s.twimg.com/a/1281028705/images/default_profile_6_normal.png',
				'loadingText' : "Loading tweets..."
			}
			$('#tuw_div').tuw(settings);
		});
	</script>
<!-- This is it -->

Here's the options list for customization

Option Description
username [string] Username of the twitter user.
simpletheme [boolean] Simpler, more minimalist and static version of the script.
updates [integer] Number of statuses to be got - Default '5'
list [string] List name of the selected username
query [string] Search query
color [string] The color scheme - blue, db, grey, green, orange, red' or bw - Default 'blue'
openbox [boolean] The box is open or close at the beginning - Default 'true'
icon [integer] You can change the icon in the header of the widget from 1 to 4 - Default '1'
autonext [boolean] Option for opening sliding to older tweets automatically. - Default 'false'
duration [integer] The time between auto-scrolls in milliseconds - Default '5000'
linksinnewpage [boolean] Option for opening all links on the widget in the same page or in a new one. - Default 'false'
showavatar [boolean] Option for displaying user's avatar or not - Default 'true'
showusername [boolean] Option for displaying username in the header or not - Default 'true'
showdate [boolean] Option for displaying tweets' dates or not - Default 'true'
singlepostmode [boolean] In this mode, only the last update will be displayed - Default 'false'
effect [string] The effect used for opening and closing the box. - slide, fade or none - Default 'slide'
